# AV Project Calculator MVP v0.3

Веб-приложение для инженерно-управленческого расчёта AV/мультимедийных проектов на базе спецификации оборудования и прайс-листов.

## Запуск

```bash
npm install
npm run dev
```

Для проверки production-сборки:

```bash
npm run build
npm run preview
```

## Что реализовано

- Импорт спецификации из Excel.
- Отдельный импорт прайс-листов из Excel в базу оборудования.
- База прайсов с быстрым поиском по модели, характеристикам и поставщику.
- Добавление позиции из прайса в спецификацию.
- Спецификация с редактированием полей, сроков, цены и критичности.
- Справочник этапов работ.
- Матрица применимости этапов по категориям оборудования.
- Нормативы трудоёмкости по категориям.
- Режим расчёта работ по часам или процентом от оборудования.
- Автоматическое разворачивание оборудования в этапы работ.
- Расчёт СМР, ПНР, проектирования, длительности и стоимости работ.
- Сводка по инсталляциям.
- Укрупнённый план-график.
- Автоматический список рисков.
- Экспорт Excel с листами: спецификация, прайс-база, справочники, матрица этапов, расчёт работ, инсталляции, план-график, риски.
- Сохранение состояния в localStorage браузера.

## Правила для прайс-листов

Одна строка = одна товарная позиция. Нельзя использовать объединённые ячейки, промежуточные итоги, многострочные шапки и произвольные блоки.

Рекомендуемые поля:

- Категория оборудования
- Тип оборудования
- Наименование оборудования
- Производитель
- Модель
- Описание
- Ед. изм.
- Цена за единицу
- Валюта
- Поставщик
- Срок поставки, календарные дни
- Статус
- Дата обновления
- Ключевые характеристики

## Логика расчёта

Трудоёмкость = Количество × Норма × Коэффициент сложности × Коэффициент риска × Коэффициенты проекта.

Длительность = Трудоёмкость / Количество специалистов / Рабочих часов в день × Запас по срокам.

Стоимость работ считается либо по часам и ставке роли, либо через процент от стоимости оборудования, если в справочнике категории выбран режим `percent`.

СМР начинается после готовности площадки и поставки оборудования. ПНР начинается после СМР. Передача начинается после ПНР.

## Ограничения MVP

- Хранилище пока локальное: данные сохраняются в браузере.
- Google Drive / папка-хранилище пока не подключены как backend-интеграция.
- Gantt-график пока табличный, без визуальной временной шкалы.
- Excel-импорт работает по названиям колонок; для стабильной работы лучше использовать утверждённый шаблон прайса.
