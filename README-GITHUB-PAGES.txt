Готовая статическая сборка AV Project Calculator для GitHub Pages.

Как загрузить:
1. Распакуйте этот ZIP.
2. Загрузите файлы из архива в корень GitHub-репозитория или в ветку GitHub Pages.
3. В репозитории откройте Settings -> Pages.
4. Выберите Source: Deploy from a branch.
5. Выберите нужную ветку и папку /root.

Важно: index.html и папка assets должны лежать рядом в корне публикации.
